Loop structure:
```java
ZoneId KST = ZoneId.of("Asia/Seoul");
LocalTime timeFrom = LocalTime.parse(request.timeFrom() != null ? request.timeFrom() : "09:00");
LocalTime timeTo = LocalTime.parse(request.timeTo() != null ? request.timeTo() : "12:00");

List<LocalDate> dates = request.from().datesUntil(request.to().plusDays(1)).toList();
List<BacktestResult.EquityPoint> curve = new ArrayList<>();
Map<String, Double> lastPrices = new HashMap<>();

for (LocalDate date : dates) {
    List<String> symbols = symbolResolver.apply(def, date);
    for (String symbol : symbols) {
        List<MarketBarIntraday> bars = loadBars(symbol, date, timeFrom, timeTo, KST);
        if (bars.isEmpty()) continue;

        double[] closes = bars.stream().mapToDouble(b -> b.getClose()).toArray();
        Double[] volumes = bars.stream().map(b -> b.getVolume() != null ? b.getVolume().doubleValue() : null).toArray(Double[]::new);

        for (int i = 0; i < bars.size(); i++) {
            MarketBarIntraday bar = bars.get(i);
            LocalDateTime barDt = bar.getTs().atZone(KST).toLocalDateTime();
            Signal sig = ruleEvaluator.evaluate(def, symbol, closes, volumes, i);
            // apply signal to ledger (BUY/SELL logic same as daily loop)
            lastPrices.put(symbol, closes[i]);
            curve.add(new BacktestResult.EquityPoint(barDt, ledger.equity(lastPrices)));
        }
    }
}
return buildResult(request.initialCash() != null ? request.initialCash() : 10_000_000.0, ledger, curve);
```

loadBars() private method:
1. Compute Instant range: date start of day KST → date end of day KST
2. Call findBySymbolAndRange(symbol, fromInstant, toInstant)
3. Filter by KST LocalTime in [timeFrom, timeTo] inclusive
4. If result empty → call yahooClient.fetchIntradayForDateRange(symbol + ".KS", date, date)
5. Filter fetched bars by time window
6. Save to DB via intradayRepository.saveAll (convert IntradayBar → MarketBarIntraday)
7. Return filtered bars

buildResult() private method — computes all stats:
- maxDrawdown: iterate curve tracking peak
- winRate: count SELL trades with pnl > 0 / total SELL trades
- sharpeRatio: per-bar returns (curr-prev)/prev, mean/stdDev * sqrt(9000)
- sortinoRatio: same returns, downside-only deviation * sqrt(9000)
- profitFactor: sum pnl>0 / abs(sum pnl<0) for SELL trades
- drawdownSegments: computeDrawdownSegments(curve)
- All division-by-zero cases return 0.0 (or MAX_VALUE for profitFactor with no losses)

computeDrawdownSegments():
```java
private List<BacktestResult.DrawdownSegment> computeDrawdownSegments(List<BacktestResult.EquityPoint> curve) {
    List<BacktestResult.DrawdownSegment> segments = new ArrayList<>();
    if (curve.isEmpty()) return segments;
    double peak = curve.get(0).equity();
    LocalDateTime ddStart = null;
    for (BacktestResult.EquityPoint p : curve) {
        if (p.equity() > peak) {
            if (ddStart != null) {
                segments.add(new BacktestResult.DrawdownSegment(ddStart, p.datetime()));
                ddStart = null;
            }
            peak = p.equity();
        } else if (p.equity() < peak && ddStart == null) {
            ddStart = p.datetime();
        }
    }
    if (ddStart != null) {
        segments.add(new BacktestResult.DrawdownSegment(ddStart, curve.get(curve.size()-1).datetime()));
    }
    return segments;
}
```
