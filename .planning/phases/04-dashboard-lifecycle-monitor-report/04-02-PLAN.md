- `TradingRule` has `status` field (String). Valid progression: DRAFT ‚Üí PAPER_LIVE ‚Üí PAUSED ‚Üí PAPER_LIVE (resume)

- `PaperRuleService.normalizeStatus` currently only allows DRAFT/ACTIVE/PAUSED ‚Äî needs extension to PAPER_LIVE and BACKTESTED
- Existing `PaperRuleController` handles CRUD; lifecycle endpoints go in a new `PaperLifecycleController`
</context>
