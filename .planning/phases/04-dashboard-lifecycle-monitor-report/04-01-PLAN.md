package com.graphify.trading.paper.dto;

import java.util.List;

public record PaperDashboardDto(
    double cash,

    double todayRealizedPnl,
    int activePaperLiveRuleCount,
    List<PaperPositionItem> positions

3. Create `backend/src/main/java/com/graphify/trading/paper/PaperDashboardService.java`:
- `@Service @Transactional(readOnly=true)`
- `getDashboard(Long userId)`:
  - Load account via `accountRepo.findByUserId(userId)` → if absent return zero dashboard
  - Load positions via `positionRepo.findByAccountId(accountId)`
  - For each position: get latest close from `intradayRepo.findBySymbolAndIntervalOrderByTsAsc(symbol,"5m")`, take last bar's close; fallback to avgPrice
  - Compute `marketValue = qty * markPrice`, `unrealizedPnl = marketValue - (qty * avgPrice)`, `unrealizedPnlPct = unrealizedPnl / (qty * avgPrice) * 100`
  - `totalEquity = cash + sum(marketValue)`
  - Today's realized PnL: filter `tradeRepo.findByAccountIdOrderByTradedAtDesc(accountId)` to trades where `tradedAt >= today midnight KST` and `side=SELL`; sum `pnl`
  - `activePaperLiveRuleCount`: `ruleRepo.findByUserIdAndModeOrderByUpdatedAtDesc(userId,"PAPER").stream().filter(r->"PAPER_LIVE".equals(r.getStatus())).count()`

4. Create `backend/src/main/java/com/graphify/trading/paper/PaperDashboardController.java`:

- `@GetMapping` → calls `service.getDashboard(HistoryService.requireCurrentUserId())`
- Returns `ApiResponse<PaperDashboardDto>`
