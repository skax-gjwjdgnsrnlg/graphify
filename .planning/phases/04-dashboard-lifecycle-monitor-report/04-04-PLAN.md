2. Create `PaperReportService.java`:
- `@Service @Transactional(readOnly=true)`
- `getReport(Long userId)`:
  - Load account via `accountRepo.findByUserId(userId)`; if absent → return empty ReportDto (zero values, empty curve)
  - Equity curve: `snapshotRepo.findByAccountIdOrderByTsDesc(accountId)` → reverse → last 30 days → map to `EquityPoint(ts.toString(), equity.doubleValue())`
  - If fewer than 2 snapshots → return partial report (no ratio computation)
  - `totalReturn`: (last equity - first equity) / first equity * 100
  - `maxDrawdownPct`: iterate equity[] computing running peak; MDD = max((peak - equity) / peak * 100)
  - Trade stats: `tradeRepo.findByAccountIdOrderByTradedAtDesc(accountId)` — filter SELL trades → `winTrades = count where pnl > 0`, `winRate = winTrades / totalSellTrades * 100`
  - `sharpeRatio` / `sortinoRatio`: compute from daily returns (group snapshots by date, compute daily % change) — use same formula as IntradayBacktestEngine: Sharpe = mean(returns) / std(returns) * sqrt(252), Sortino = mean(returns) / downside_std * sqrt(252). If fewer than 5 data points → return 0.0
