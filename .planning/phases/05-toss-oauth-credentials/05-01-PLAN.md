1. Create `TossCredentialRequest.java` record:
```java
public record TossCredentialRequest(String clientId, String clientSecret) {}
```

2. Create `TossCredentialStatusDto.java` record:
```java
public record TossCredentialStatusDto(
    boolean configured,
    boolean tokenValid,
    Instant tokenExpiresAt
) {}
```

3. Create `TossCredentialService.java`:
- `@Service @Transactional`
- `saveCredentials(Long userId, String clientId, String clientSecret)`:
  - findByUserId or create new TossCredential
  - encrypt both fields, clear token (force re-issue), save
- `getDecryptedClientId(Long userId)` → decrypt or throw NOT_FOUND
- `getDecryptedClientSecret(Long userId)` → decrypt or throw NOT_FOUND
- `getStatus(Long userId)` → TossCredentialStatusDto
  - configured = row exists
  - tokenValid = accessTokenEncrypted != null && tokenExpiresAt.isAfter(now + 1 min buffer)

4. Create `TossTokenManager.java`:
- `@Service`
- Inject `TossCredentialRepository`, `SecretEncryptionService`, `RestClient.Builder` (Spring 6) or `RestTemplate`
- `ensureValidToken(Long userId) → String` (returns decrypted access_token):
  - Load credential; if none → throw GraphifyException NOT_FOUND
  - If token valid (not expired - 10 min) → decrypt + return
  - Else → `issueToken(userId, credential)` → save + return
- `issueToken(TossCredential cred)`:
  - Decrypt clientId + clientSecret
  - POST to `https://openapi.tossinvest.com/api/v1/oauth2/token` with body: `grant_type=client_credentials&client_id=...&client_secret=...` (form-urlencoded)
  - On success: encrypt access_token, set tokenExpiresAt = now + expires_in seconds, save
  - On HTTP error: throw GraphifyException with BAD_GATEWAY
- `@Scheduled(fixedDelay = 300_000)` `refreshExpiringSoon()`:
  - Find all credentials where tokenExpiresAt between now and now+10min
  - For each: call issueToken (refresh)
  - Log warn if refresh fails, continue (swallow exception per credential)

5. Create `TossCredentialController.java`:
- `@RestController @RequestMapping("/api/v1/toss/credentials")`
- `GET /status` → `ApiResponse<TossCredentialStatusDto>`
- `POST /` → save clientId + clientSecret → `ApiResponse<TossCredentialStatusDto>`
- `POST /token/refresh` → manually trigger ensureValidToken → `ApiResponse<TossCredentialStatusDto>`

6. `./gradlew compileJava`
