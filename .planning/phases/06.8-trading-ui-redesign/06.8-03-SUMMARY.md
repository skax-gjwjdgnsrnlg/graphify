## TradeTable + Native Table Pattern

For tables where e2e tests rely on `tr`/`td` DOM selectors, wrap with TradeTable but keep native `<table>` inside:

```tsx
<div className="rounded-lg border border-trade-hairline bg-trade-surface overflow-hidden">
  <div className="overflow-x-auto">
    <table className="w-full text-sm">
      <thead>
        <tr className="border-b border-trade-hairline text-trade-muted text-left">
          <th className="px-4 py-3 font-medium font-trade-sans">...</th>
        </tr>
      </thead>
      <tbody>
        <tr className="border-b border-trade-hairline/50 hover:bg-trade-elevated transition-colors">
          <td className="px-4 py-3 text-trade-muted font-trade-mono">...</td>
        </tr>
      </tbody>
    </table>
  </div>
</div>
```

Selected row: `border-l-2 border-l-trade-primary bg-trade-elevated`
PnL cell: `text-trade-up` (>0) / `text-trade-down` (<0) / `text-trade-muted` (=0) + `font-trade-mono`
